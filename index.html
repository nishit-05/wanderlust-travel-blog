About this project

Built for AWS.
Designed for the road.

This website was built as part of the AWS Cloud Fundamentals course offered through Accenture's partnership with Udacity. The project covers deploying a static website on Amazon S3, securing it with IAM bucket policies, and distributing it globally using Amazon CloudFront.

The site itself is a travel journal concept — a place to document destinations, write up experiences from the road, and share stories from different parts of the world. All pages are static HTML, CSS and JavaScript with no backend or server-side code, making it a perfect fit for S3 hosting.
